Описание проекта - прототип базы данных, работающей в оперативной памяти, с одной таблицей "Студенты". Реализован на Python с консольным интерфейсом для взаимодействия с пользователем.
Структура проекта -
│
├── src/
│ └── db/
│ └── backend/
│ │ ├── **init**.py - делают папки видимыми для Python
│ │ └── memory.py - здесь хранятся данные и создаются записи, проверяется на ошибки ввода
│ │
│ ├── **init**.py - делают папки видимыми для Python
│ ├── **main**.py - запускает программу, вызывает tui
│ └── tui.py - общение с пользователем, вызывает функции из memory, принимает ввод пользователя и показывает записи на экране
│
└── README.md
Как запустить программу - python -m src.db
